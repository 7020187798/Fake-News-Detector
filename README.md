# Fake News Detection using Machine Learning

 # Project Overview
This project detects whether a news headline is **Fake or Real** using Machine Learning.

# Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)

# How it Works
- Data preprocessing using pandas
- Text converted into numerical form using TF-IDF
- Logistic Regression model used for classification

# Model Accuracy
Achieved accuracy of around 47.83% on test data.

# How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the file:
   python news.py

3. Enter any news headline to test prediction

# Example
Input: Aliens spotted in Mumbai  
Output: Fake

Input: Government announces new policy.
Output: Real
